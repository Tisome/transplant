# 1.4 论文组织结构

全文共分为七章。第一章为绪论，介绍研究背景、研究现状、主要研究内容和论文结构。第二章为超声波时差测量原理与系统需求分析，说明时差法测量原理、传播时间关系、流速流量换算和系统设计需求。第三章为系统总体方案与数据流程设计，说明 FPGA 与 MCU 的分工、信号流和特征数据包边界。第四章为基于 FPGA 的超声波时差测量模块设计，重点介绍激励采样时序、缓存处理、相关计算、峰值搜索、特征提取和接口输出。第五章为 MCU 端时间差解算与流量输出实现，说明特征解析、抛物线插值、`dt` 解算、流量换算和结果稳定化处理。第六章为系统仿真与验证分析，介绍 MATLAB 仿真平台、时差与流量正确性验证、噪声鲁棒性分析和板级联调流程。第七章为总结与展望。
