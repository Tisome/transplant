# 1.4 论文组织结构

全文共分为七章。第一章为绪论，主要介绍研究背景与研究意义、国内外研究现状、本文主要研究内容及论文整体结构；第二章为超声波时差测量原理与系统需求分析，主要介绍时差法测量原理、顺流与逆流传播时间关系、时间差与流速流量的换算关系以及系统设计需求；第三章为系统总体方案与数据流程设计，主要介绍系统总体架构、FPGA 与 MCU 的任务划分、信号流与数据流设计以及特征包格式设计；第四章为基于 FPGA 的超声波时差测量模块设计，重点介绍前级激励与采样时序、相关处理、峰值搜索、特征提取及接口输出设计；第五章为 MCU 端时间差解算与流量输出实现，主要介绍特征包解析、抛物线插值、传播时间差解算、流量换算以及结果平滑与稳定输出；第六章为系统测试与验证方法，主要介绍 MATLAB 仿真平台构建、时差提取正确性验证、流量输出正确性验证、噪声条件下的鲁棒性分析以及板级联调验证；第七章为总结与展望，对全文工作进行总结，并对后续研究方向进行展望。
