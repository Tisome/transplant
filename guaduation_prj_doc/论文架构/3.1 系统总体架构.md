# 3.1 系统总体架构

本系统以超声波时差测量为核心，面向液体流量测量需求，构建了由超声换能器、前端激励与接收电路、模数转换模块、FPGA 处理模块、MCU 解算与控制模块以及显示通信模块组成的测量系统。系统设计目标是在保证时间差提取精度的基础上，实现稳定、实时且具有工程可实现性的流量输出。

在整体结构上，FPGA 位于前端处理链核心位置，负责产生换能器激励时序、完成采样同步，并对回波采样结果进行高速预处理；MCU 位于后端，负责基于 FPGA 提供的关键特征完成时间差精细解算、流量计算、参数管理及结果输出。激励与接收电路、ADC 以及显示通信部分分别承担信号驱动与调理、模数转换以及结果呈现等辅助任务。由此，系统形成了由“前端测量”与“后端解算”构成的分层结构。

从工作流程看，系统由 FPGA 发出激励控制信号驱动换能器发射超声波，并在相应时刻触发 ADC 对接收回波进行采样。采样结果在 FPGA 内部完成相关处理与特征提取后，以固定格式送入 MCU，后者再完成时间差求取及流量输出。根据现有设计，系统以前后两个 `4 ms` 子过程构成一个完整测量周期，即约 `8 ms` 完成一组顺逆流数据的获取与处理。该架构既发挥了 FPGA 在高速并行处理和确定性时序方面的优势，又保留了 MCU 在算法组织和系统管理方面的灵活性。后续各节将在此基础上进一步说明任务划分、信号与数据流设计以及关键接口组织方式。
