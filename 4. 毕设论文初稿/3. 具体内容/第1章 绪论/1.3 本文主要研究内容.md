# 1.3 本文主要研究内容

本文面向超声波时差测量中微小传播时间差的提取需求，构建了一套基于 FPGA 与 MCU 协同实现的测量模块方案。全文围绕测量原理、前级特征提取、后级时间差解算和系统验证展开，形成“回波采样特征化 - 特征包传输 - `dt` 解算 - 流量输出 - 仿真与联调验证”的技术链路。

首先，针对超声波时差测量中的基本问题，本文分析了顺流与逆流传播时间差同流速、流量之间的关系，并结合实际测量需求，对系统的时间分辨率、实时性和抗干扰能力进行了讨论。在此基础上，明确了时差测量模块在整个测量链路中的核心地位，为后续系统设计提供了理论依据。

其次，针对前级高速信号处理需求，本文设计了基于 FPGA 的时差特征提取模块。该模块完成超声信号采样、相关处理、峰值搜索和特征输出，将整段采样波形压缩为后级解算所需的关键特征，从而降低跨平台传输和后级计算压力。

再次，针对后级时间差解算与结果输出需求，本文设计了 MCU 端处理方法。该部分根据 FPGA 输出的峰值索引及邻域特征数据，采用抛物线插值细化峰值位置，完成传播时间差 `Δt` 解算，并进一步实现流量换算、异常剔除和稳定输出。

最后，针对当前阶段难以开展完整装管实测的实际条件，本文构建了 MATLAB 仿真与板级联调相结合的验证方案。仿真部分用于建立时间差模型、流量变化模型及噪声扰动条件，对时差提取精度和算法鲁棒性进行分析；联调部分用于验证 FPGA 特征输出、MCU 数据接收及后级解算链路的可实现性，从而对所设计模块进行较为完整的验证。
