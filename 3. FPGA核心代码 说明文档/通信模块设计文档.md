# 通信模块设计文档

模块设计目标

本模块是一个SPI主机控制器，将FPGA内部处理完的16位索引数据通过SPI发送给外部芯片GD32。

## 二、架构设计

该模块的协议标准为SPI Mode 0，空闲低电平，上升沿发送，下降沿采样。由输入信号start脉冲触发一次完整的传输。发送移位寄存器（tx_shift）锁存并行数据并按位输出到MOSI，时钟发生器（spi_sclk_reg）通过每个clk上升沿对信号进行翻转产生50MHz的时钟信号。

## 三、详细逻辑分析

启动阶段：当start为高电平时拉低片选信号激活从机；将并行输入数据存入缓冲寄存器；初始化计数器；置位busy，开启时钟。

Sclk时钟上升沿逻辑：将tx_shift[bit_cnt]的内容移动到spi_mosi线上。

复位阶段：当bit_cnt == 0且最后一拍采样完成后：拉高片选信号；产生done信号脉冲；清零busy。
