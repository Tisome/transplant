# PWM模块设计文档

模块设计目标

本模块用于实现基于FPGA的信号发生器，通过控制输出信号的占空比实现对超声波换能器器件的控制

## 二、架构设计

为了尽可能实现解耦，本设计将PWM系统分为控制单元和数据单元。其中，控制单元的功能为倍频生成时间系统时间、生成并管理数据寻址、输出工作控制信号和工作状态，数据单元的功能为存储预设波形数据、根据控制信号生成并管理PWM波形。

## 三、详细逻辑分析

### 3.1 控制单元逻辑

1. 分频计数器：将系统时钟（50Mhz）倍频为clk_out1（100Mhz）和clk_out2（200Mhz），之后以clk_out1为基准分频给计数器cnt_4ms，实际上该计数器技术范围为0-200,000。单次循环时间为2ms，每次达到100,000时翻转channel_sel，实现了每4ms为循环的PWM输出的频道选择

2. 地址计数器：对于地址计数器，其所用的时钟为clk_out2（200Mhz）。当cnt_4ms为1时，pwn_en变为1，从下个时钟周期开始addra自增1。直到addra等于499时，pwm_en变为0，同时addra变为0，从该时钟周期开始输出3个时钟周期（约15ns）的ad_start信号，用以提示ADC开始工作

### 3.2 存储读取逻辑

该模块采用Block RAM资源构建只读存储器（ROM），通过加载.coe配置文件进行初始化。仿真时可以采用这种方式，用.coe文件在ROM中存储了离散化的波形特征点。由于BRAM是同步器件，地址addra发出后，数据douta会根据配置产生1-2个时钟周期的延迟，需要通过时序约束使得数据能够被正确采样

### 3.3 数据路径逻辑

当pwm_en和channel_sel是均为高电平时，从ROM中读取的数据输出到pwm_p和pwm_n，当pwm_en为高电平、channel_sel为低电平时，从ROM中读取的数据输出到pwm2_p和pwm2_n。这里会有一个时间周期的延迟，当寻址地址信号addra为499时，在下个周期立刻将输出数据拉平到0，这样保证不会出现时序错误。
