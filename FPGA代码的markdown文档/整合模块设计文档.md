# 整合模块设计文档

模块设计目标

本模块通过SPI控制外部超声波换能器，随后接收返回的AD信号（每组2000个），转换数据精度至16bit，将每组数据与已经保存的参考数据进行卷积操作，从中筛选出最大卷积数据的索引并通过SPI发送给上位机（GD32）。

## 二、架构设计

该模块分为时钟管理模块、pwm输出控制模块、SPI增益控制模块、AD数据采集模块、卷积算法模块、峰值检测模块、通信上报模块。

## 三、系统工作流程

## 1. 启动与发射：外部start信号触发PWM模块，开始向换能器发送激励信号 。

## 2. 同步触发：发射结束瞬间，PWM模块发出ad_start脉冲 。

## 3. 采集与存储：AD模块响应脉冲，抓取2000个ADC样本并存入缓存 。

## 4. 降速读取与运算：数据以2MHz的速率被读取并输入卷积模块，计算回波匹配度 。

## 5. 峰值检测：timeget在运算过程中动态更新最大值及其索引 。

## 6. 数据外传：一帧处理结束后，Top驱动SPI通信模块将最终的索引结果发送给MCU（GD32） 。
